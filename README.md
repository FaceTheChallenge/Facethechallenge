Facethechallenge
================

Repositorio del proyecto Face the challenge

Para poder hacer funcionar el proyecto se necesita tener instalados los
siguientes módulos de Node.js:



 - ejs
 - express
 - nano
 - restler
 
 Y las siguientes funciones que deberán de estar incluidas en /publics/js/funcionesExternas.js:
 
  - md5 [https://github.com/kvz/phpjs/blob/master/functions/strings/md5.js]
  - utf8_enconde [https://github.com/kvz/phpjs/blob/master/functions/xml/utf8_encode.js]
